# VtlEditor component

## Props

| Props |  Type  | Default value | Required | Description         |
| :---: | :----: | :-----------: | :------: | ------------------- |
| focus |  bool  |     false     |    -     | Is the editor focus |
| theme | string |   "vs-dark"   |    -     | Editor theme        |

## Styles

**VtlEditor** component has for class `vtl-editor`.
